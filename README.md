# expo-detox-example


test with

* Expo SDK 38
* Detox 17.3
* Mocha 8

#### Install it
```
yarn install
```

#### Run it

download ipa from expo client -> ./bin/Expoment.app
```
yarn build-testApp
```

run your first test
```
yarn detox
```

Detox for more test informations:

https://github.com/wix/Detox
